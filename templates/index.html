Welcome to the Bitcoin Prediction API

About the API

This API provides predictions for Bitcoin prices using a machine learning model. You can also access historical data, retrain the model, and evaluate its performance.

Available Endpoints

  • POST /predict - Predict Bitcoin price for a given date.
  • POST /train - Retrain the model with the current dataset.
  • POST /history - Get predictions for a range of dates.
  • GET /metrics - Get model performance metrics (MSE).
